Component: EnergyPlus
Category: HVACTemplate
CR Number: 7,166
Entered On: 11/16/2006 12:47:52 PM
Synopsis for Users: HVACTemplate - VAV system sizing min flow rate fixed at 0.3 can cause heating or preheat undersizing.
Further Details for Users: Reference: Enhancement List Item: HVAC_AirLoop_2009_16
Work Around: Review the component sizing results from the eio output file. Find the zone design air flow rates, the system design air flow rate, and the zone min fractions. From these values compute the max and min system supply flow rates. For zones with REVERSE ACTION heating controls, use the max supply flow rate as the min. The min system supply flow rate is the flow rate used to size central heating and preheat coils.

Take the expidf file from the original run, and edit the SYSTEM SIZING object. Put the ratio of system min flow/max flow in the "minimum system air flow ratio" field.
